Key Implementation Goals

Mobile-First

Designed mobile baseline styles first (under 992px) with toggled overlays. Used a media query at 992px to seamlessly upgrade to a horizontal desktop navbar layout.

Accessibility (A11y)

Features semantic HTML structuring (<header>, <nav>, <button>). Complete keyboard navigation with custom active focus outlines.

Smooth Micro-Interactions

Animated menu button (hamburger to cross) and smooth sliding nav transitions. Respects system level prefers-reduced-motion preferences.
